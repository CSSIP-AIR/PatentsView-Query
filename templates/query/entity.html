The PatentsView Query Tool

The PatentsView Query Tool provides an easy to use web interface for querying data on patents, applications, inventors and assignees from the PatentsView database. To begin your search, click on one of the primary output entities: patent, inventor, or assignee. Click through the steps of the query to select search criteria, identify output fields, and sort and limit results. You will receive results in the designated format via email within 24 hours. Similar queries can also be completed using the PatentsView API.
