Changing Category

Warning, changing the category will reset the query input values you've provided. Click Ok to change the category, or Cancel to keep your current selections.

The PatentsView Query Tool provides an easy to use web interface for querying data on patents, inventors, and assignees from the PatentsView database.

To begin your search, select patents, inventors, or assignees in Step 1 below. This will determine the primary field in your output dataset.

Use the “quick search” option below to search across patent titles, abstracts, inventor and assignee names. Patents will be the primary entity in the output dataset. Boolean operators can be used.

You will receive output data files via email within 24 hours of submitting a search.
