What is a Patent Class?

Patents are classified by three distinct schemes in the PatentsView data visualization tool:

  • US patent classification (USPC)
  • The National Bureau of Economic Research (NBER) (Hall, Jaffe, Trajtenberg, 2001) technology categories
  • Cooperative patent classification (CPC)

All USPC, NBER, and CPC patent classes appearing in the visualization tool represent the current patent class, unless otherwise noted.

How to Search for a Patent Class

Select a patent classification system tab above, then find the class symbol or number in the list. Enter it or a keyword into the filter. For each patent, the USPC primary main class, the NBER primary subcategory, and all CPC subsections and classes are searchable. All classifications are current class.
