The PatentsView Query Tool

The PatentsView Query Tool provides an easy to use web interface for querying data on patents, inventors, assignees, patent classes, and locations from the PatentsView database.

To begin your search, select data output fields from the tables below. Then, click through the steps to select search criteria and sort and limit results. You will receive results in the designated format via email within 24 hours.

Similar queries can also be completed using the PatentsView API.
